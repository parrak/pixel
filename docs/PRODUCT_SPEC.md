# Product Spec

## North Star

Build a synthetic ASC/surgical RCM copilot that helps reviewers identify revenue-cycle work, understand why it matters, see cited evidence, and generate reviewer-safe drafts without automating final coding, billing, compliance, or payer decisions.

## Users

- Coder
- Biller
- Denial specialist
- Auth specialist
- Manager

## Core Product Areas

- AI Coding Copilot
- A/R Flagging Copilot
- Denials and Appeals Copilot
- Workflow Assistant
- Payer Pattern Intelligence
- Work Queue Prioritization
- Reviewer Packets and Drafts
- Streamlit Workbench
- Copilot Eval Harness

## Hard Product Boundaries

- Synthetic data only
- Human review required everywhere
- Deterministic rules authoritative
- No external APIs by default
- No real payer submission or automation
- No PHI

## Primary Outputs

- Evidence-cited opportunities
- Priority-scored work queue items
- Role-aware workflow suggestions
- Reviewer packets
- Reviewer-safe drafts
- Aggregate payer intelligence summaries
- Eval metrics and manager summaries
