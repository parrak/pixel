# Compliance Guardrails

ASC RCM Command Center Lite is a synthetic ASC and surgical revenue cycle
prototype. It is designed for workflow demonstration, experimentation, and
software development only.

## Non-Clinical Prototype

This project is not clinical software. It must not be used to diagnose, treat,
monitor, triage, or otherwise guide care for any patient.

## No Coding Or Billing Advice

The prototype does not provide medical coding advice, billing advice,
reimbursement advice, payer policy interpretation, or compliance determinations.
Any examples that reference coding, billing, denials, documentation, charge
capture, or revenue cycle operations are synthetic workflow examples only.

## No Real Payer Connectivity

The project is not connected to real payers, clearinghouses, EHRs, practice
management systems, revenue cycle systems, or patient data sources. Any payer,
claim, encounter, case, denial, or payment data used by the prototype must be
synthetic test data.

## Legacy Clinical RI Examples

Clinical RI Lite material should be treated as legacy clinical revenue integrity
examples. Existing AKI, sepsis, respiratory, and related Clinical RI Lite tests
should remain in place when present and should continue to validate the legacy
examples during the transition to ASC RCM Command Center Lite.
